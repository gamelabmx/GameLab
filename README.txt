Publicación rápida en GitHub Pages:
1. Crea un repositorio nuevo en GitHub.
2. Sube todos estos archivos a la raíz del repositorio.
3. Verifica que el archivo principal se llame exactamente: index.html
4. En GitHub, entra a Settings > Pages.
5. En "Build and deployment", selecciona "Deploy from a branch".
6. Elige la rama principal (main) y la carpeta raíz (/root).
7. Guarda los cambios.
8. GitHub te dará la URL pública del sitio.

Nota:
El formulario de contacto es visual/estático. Si deseas, después se puede conectar a:
- Formspree
